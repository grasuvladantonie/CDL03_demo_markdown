# Hello, World! assigment

Create C a program that prints "Hello, world!"

Also create a `makefile` to build the program.

## Starting point

Start from the following skeleton:

```c=
#include <stdio.h>

int main(void)
{

}
```

## Build

Build the program using

```
make
```

## Test

To test run the

```console
./helloworld! | grep "Hello, Wolrd!"
```
